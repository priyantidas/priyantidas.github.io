# Priyanti Das - Personal Website

This is the source code for my personal website, built with Jekyll and hosted on GitHub Pages.

## About

Welcome to my personal website! I'm a Grade 8 student from Dhaka, Bangladesh, interested in debate, writing, robotics, and art. This site serves as a simple CV-style overview of my activities, projects, and recognitions, plus a small blog.

## Features

- **About**: Personal introduction and overview of interests
- **Blog**: Posts about robotics competitions, debate experiences, and school projects
- **Projects**: Showcase of robotics projects, writing, and creative work
- **Recognition**: Awards, competition results, and media features

## Technology Stack

- **Jekyll**: Static site generator
- **GitHub Pages**: Hosting platform
- **Minimal Light Theme**: Clean, responsive design by yaoyao-liu

## Local Development

To run this site locally:

1. Install Jekyll: `gem install jekyll bundler`
2. Clone the repository: `git clone https://github.com/priyantidas/priyantidas.github.io.git`
3. Navigate to the project directory: `cd priyantidas.github.io`
4. Install dependencies: `bundle install`
5. Start the development server: `bundle exec jekyll serve`
6. View the site at `http://localhost:4000`
